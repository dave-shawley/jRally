jRally is a Java integration to the Rally Agile project management suite.

I have been using Rally for a while at work now and find that while its user interface is quite pleasant to use from a web browser, it is anything but usable from a mobile device. There are also several problems with its ability to print user stories nicely. To make a long story short, I decided to write some utilities to make my life a little easier and this project is the result of that effort.

This is a first effort in a number of arenas for me. First of all, I have been a long time Subversion user so I decided to try something new and host this project on github and delve into the world of distributed version control systems. So, you can find the most recent version of this project on github until either (1) they start charging or (2) I give up and go home.

The next first for me is actually writing a Java project. I am a C++ programmer by choice and just started dabbling in Java recently so be kind and accept all of the warts and ugliness that you are likely to find herein.
